# Product Context

## Purpose
Traditional printed wedding invitations lack interactivity, real-time updates, and logistical tracking. "Prabal Ki Shaadi" solves this by centralizing the guest experience and the family's event planning into a single web application.

## Problems Solved
- **For Guests**: Eliminates confusion regarding event schedules, venues, and travel routes. Provides an immediate, trackable way to RSVP.
- **For the Core Team**: Replaces scattered WhatsApp groups and Excel sheets with a centralized dashboard tracking who is coming, what tasks need to be done, and how much money is being spent.

## Expected User Experience
- **Guests**: Arrive at a stunning, slow-animated hero section with a live countdown and falling golden dust. Navigation is simple (Home, Events, Travel, RSVP). The RSVP process is frictionless.
- **Members (Management)**: Access a private `/login` route. Upon entering the dashboard, they see data tailored to their role. Data entry (adding expenses, delegating tasks, adding guests manually) is quick, responsive, and visually cohesive with the main site's luxurious theme.
