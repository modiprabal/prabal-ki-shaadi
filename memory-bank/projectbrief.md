# Project Brief

## Project Overview
"Prabal Ki Shaadi" is a premium, dual-purpose wedding platform built for Prabal and Shreya's wedding (Dates: 14-15 December 2026, Lalitpur, India).

## Core Goals
1. **Digital Invitation (Guest View)**: Provide a highly animated, luxurious, and mobile-friendly invitation experience. Guests can view the itinerary, travel details, and submit their RSVPs.
2. **Management Portal (Member View)**: Provide a secure dashboard for the core wedding team (Couple, Admins, Coordinators) to manage the budget, guest list, and task delegation.

## Key Requirements
- **Aesthetic**: Modern minimalist, luxurious, using Framer Motion for smooth entrances, glassmorphism, and a primary color palette of cream, gold, and emerald.
- **Role-Based Access Control (RBAC)**: Distinct permissions for everyday guests vs. management roles (Couple, Admin, Coordinator).
- **Responsive Design**: Mobile-first priority, ensuring the invitation looks spectacular on phones while remaining functional on desktops.
- **Future Database Integration**: Currently powered by a robust mock data store, designed to be seamlessly hot-swapped with Supabase (PostgreSQL) in the final phase.
